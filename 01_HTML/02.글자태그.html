문단 구분 태그 : p, pre

문단을 나누는 태그로 p태그와 pre태그가 있다. p 태그는 태그 내에 줄바꿈 처리를 하고자 한다면 별도의 태그가 필요하다
** 줄바꿈 처리 **
그리고 공백은 한 개의 공백만 표시하므로 별도의 기호 문구를 사용함
-> 기호 문구를 사용하지 않는 경우 : @ @
-> 기호 문구를 사용한 경우 : @   @ 

pre태그는 시작태그에서부터 종료태그까지 입력된 내용
          그대로 표현하는 태그다.
          줄바꿈 처리, 공백이 모두 표현됨.

그 외의 글자 관련 태그

태그 없이 일반적인 글자 표현 가능

strong : 글자를 굵게 표시하는 태그 ( + 스크린리더)

em : 글자를 기울여서 표시하는

mark : 형광펜 효과와 함께 표시하는 태그

u : 글자 하단에 밑줄을 표시하는 태그

s : 취초선을 표시하는 태그

small : 글자를 작게 표시하는 태그
