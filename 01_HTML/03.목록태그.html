목록 형태로 표시해주는 태그

  • 목록을 표시 태그 : li
  • 기본 값은 순서 없는 태그(ul)

    • ul : 순서 없는 목록 태그

    • HTML
    • CSS
    • JavaScript
    • jQuery
    • HTML
      • 글자 관련 태그 : p, pre
      • 목록 관련 태그 : ul, ol
    • CSS
    • JavaScript
    • jQuery

    ol : 순서 있는 목록 태그

    1. HTML
    2. CSS
    3. JavaScript
    4. jQuery

    숫자 표시, 7부터 시작, 역순 목록 표시

    영문 표시, 7번째 알파벳 소문자 목록 표시

    아라비아 숫자 표시, 3번째 아라비아 숫자 목록 표시

    ol : 위 조건 목록 실습

    1. HTML
      1. p
      2. pre
      3. b
      4. strong
    2. CSS
      1. style
      2. width
      3. color
      4. .....
    3. JavaScript
      1. print
      2. println
      3. console
      4. ....
    4. jQuery

    dl : 설명 관련 목록 태그 (주로 용어 정의 시 사용)

    제목(용어) 입력 태그
    설명 입력 태그
    프론트엔드 영역
    HTMl, CSS, JavaScript 등의 언어 사용 웹 브라우저 화면 구성
    백엔드 영역
    Java, DB 등을 사용하여 사용자의 요청에 대한 처리 응답을 하여 서비스 제공
