window.confirm("질문내용")

confirm 호출 시 "질문 내용", 확인/취소 버튼이 표시되는 알림창표시됨
확인 버튼 클릭시 true, 버튼 클릭 시 false


[window.]prompt("질문내용")

prompt 호출 시 "질문내용", 입력 할 수 있는 "텍스트 상자", 확인/취소 버튼이 보여지는 알림창이 표시 됨
확인 버튼 클릭 시 텍스트 상자에 입력된 값을 반환
취소 버튼 클릭 시 null을 반환 (* 텍스트 상자에 값이 입력되어 있어도 취소 버튼 클릭시 null을 반환)
